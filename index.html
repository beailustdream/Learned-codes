Quem sou eu?

Sou desenvolvedora de software com experiência em análise, design e construção de aplicações escaláveis e eficientes. Ao longo da minha carreira, trabalhei em projetos web, desktop e mobile, utilizando tecnologias modernas e boas práticas de desenvolvimento. Tenho paixão por resolver problemas complexos por meio da programação, sempre buscando soluções limpas, seguras e de alto desempenho. Valorizo o trabalho em equipe, a comunicação clara e o aprendizado contínuo. Acredito que o código bem escrito é aquele que entrega valor real para o usuário e pode ser mantido com confiança. Tenho familiaridade com metodologias ágeis, versionamento com Git e integração contínua, além de atuar com testes automatizados e arquitetura de software. Meu objetivo é contribuir com projetos que impactem positivamente a vida das pessoas e continuar evoluindo como profissional.

Habilidades

  • Lógica de programação: pensar de forma estruturada e resolver problemas passo a passo.
  • Linguagens de programação: como Python, Java, JavaScript, C#, etc.
  • Versionamento de código: saber usar Git e GitHub.
  • Estruturas de dados e algoritmos: para escrever código eficiente e organizado.
  • Banco de dados: uso de SQL (MySQL, PostgreSQL) e NoSQL (MongoDB).
  • Desenvolvimento Web: HTML, CSS, JavaScript, e frameworks como React ou Angular.
  • Testes de software: escrever testes unitários, de integração e automação.
  • APIs: entender e usar APIs RESTful.
  • Controle de qualidade e debugging: saber encontrar e corrigir erros.
  • Conhecimento de sistemas operacionais: como Linux e Windows.
  • Arquitetura de software: princípios como MVC, camadas, e boas práticas de design.
