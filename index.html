EXERCÍCIO 01: TABLE BORDER TIPO TAMANHO E ALTURA.

Dica 1: Se aplica table, border, width e height, os tipos de bordas vão do 1 até o 1000, só que o mil faz a tabela desaparecer e ficar apenas um vulto cinza, então o limite recomendado é de 1 a 10!

Dica 2: Se aplica table, border-collapse, width e height, o tamanho da tabela é 100% e a altura é 100%.

Dica 3: Tr também pode se aplicar estilos, como por exemplo o background, o align e o style.

Dica 4: TBODY é o corpo da tabela, TFOOT é o rodapé da tabela, TREAD é o cabeçalho da tabela.

Dica 5: Se na tag TH, estiver usando STYLE é obrigatório usar TEXT ALIGN.

Dica 6: COLSPAN para concatenar colunas, ROLSPAN concatena linhas, para concatenar por completo deve se por o número correto de linhas ou colunas da tabela para rodar a tag.

Dica 7: A demonstração do uso do TFOOT está na tabela abaixo, ultilizada a concatenação COLSPAN no TH, é possivel ultilizar mais de um rodapé com o auxilio de encerrar e reabrir os TH da forma que for necessária.

Dica 8: Dá para alterar o negrito padrão das fontes da tabela, só mandar um TD STYLE ALINHAMENTO FONT WEIGHT e o tipo que quer mudar.

Dica 9: Para deixar o css fazer a tabela é só substituir o TABLE BORDER pro TABLE STYLE.
